# 🎨 文字調色盤 (Word Palette)

這是一個專為 國小三、四年級學生 設計的互動式 AI 語文學習工具。透過 AI 生成具備畫面感的優美句子，引導學生進行「文字與色彩」的轉化練習，並由 AI 擔任溫柔的國文老師，給予即時、具備標籤重點的溫馨點評。

### 🌟 遊戲特點
### 1. 多元化意境生成
系統會隨機從以下維度出題，確保學生有最廣闊的想像空間：

微觀世界：例如觀察露珠裡的倒影。

抽象心情：將無形的情緒轉化為可感知的文字。

奇幻想像：如雲朵上的建築或森林裡的秘密。

動態美景：捕捉大自然瞬息萬變的姿態。

嚴格限制：句子中絕對不會出現顏色名稱（如紅、藍、綠），完全交由學生觀察文字意境來「調色」。

### 2. 結構化標籤點評
老師的評語不再是冗長的段落，而是透過明確的標籤進行引導：

【亮點】：標示學生選色中「有充分表達」意境的部分，給予具體鼓勵。

【小建議】：標示顏色「未充分展現」或可以進一步嘗試的方向。

純淨排版：移除艱澀成語與多餘符號，專注於美感與直覺的對話。

### 3. 虛實整合：金句抄寫區
為了配合實體教學與學習單設計，特別開發了「抄寫功能」：

對照功能：結果頁面會清晰呈現 AI 生成的原句。

學習單對接：方便學生將優美金句抄錄至紙本學習單，達成數位體驗與手寫練習的平衡。

## 技術架構
前端介面：HTML5 / CSS3 (採用適合低年級的粉色調與大字體設計)。

後端邏輯：Google Apps Script (GAS)。

AI 引擎：支援 Gemma 4 (26B/31B) 與 Gemini 1.5 Flash 雙模型備援機制。

資料安全：透過指令碼屬性 (Script Properties) 管理 API Key。

## 部署說明
### 步驟一：環境設定
在 Google Apps Script 建立新專案。

建立 Code.gs 並貼入後端邏輯。

建立 index.html 並貼入前端介面。

### 步驟二：API 金鑰配置
點擊專案設定（齒輪圖示）。

在「指令碼屬性」中新增：

屬性：GEMINI_API_KEY

值：你的 Google AI Studio API Key。

### 步驟三：發佈
點擊「部署」>「新部署」。

類型選擇「網頁應用程式」。

存取權限設定為「任何人」。

## 教學應用建議
本工具最適合與 三、四年級自然或語文領域 結合：

課前引導：老師分享一段無顏色的文字，請學生口頭描述畫面感。

數位創作：學生操作「文字調色盤」，進行選色與理由撰寫。

實體紀錄：參考畫面的點評與金句，完成「我的文字色彩紀錄」學習單。
