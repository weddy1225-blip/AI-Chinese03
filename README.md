# 🎨 文字調色盤 (Word Palette)

這是一個專為國小四年級學生設計的 **AI 語文互動遊戲**。
透過「標籤盲選」機制，讓學生觀察形容詞如何改變故事的視覺材質與氛圍。

## 🚀 立即開始遊戲
 [👉 點擊此處開始遊戲 👈](https://weddy1225-blip.github.io/AI-Chinese03/)


### 🌟 遊戲特點
- **三種故事路線：** 神祕山洞、販賣機秘密、魔法校車。
- **材質感應系統：** 包含金屬感、透明感、復古木質感等 15 種視覺特效。
- **語文教學：** 練習形容詞擴寫與情緒感知。

### 🛠️ 技術實作
- 純 HTML5 / CSS3 (含複雜漸層特效) / JavaScript。
- 無須後端，開箱即用。
